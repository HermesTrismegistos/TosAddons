# Legend ExpPotion Gauge
レジェンド経験値ポーション（ファモカの試薬瓶）の経験値量を常時可視化します。  

## 注意事項
* クイックスロット・インベントリ・倉庫・チーム倉庫に対応しています。  
その他マーケットなどには対応していません。
## 既知の不具合
* インベントリ内の経験値バーが更新されないときがある
