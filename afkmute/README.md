# AFKMute
AFKしたいときにボタン操作で音量をミュートにできます。  
__右クリック__ でミュートONOFFを切り替えます。  
白チャで`/afkmute`でもトグルできます。  
初回起動時は左上にボタンが出ますので、左ドラッグで好きな位置に移動してください。
